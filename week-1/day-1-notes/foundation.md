  Q.1: HOW INTERNET WORK?
  ANS; The internet is a massive, global "network of networks." It works like a giant postal system for data: breaking information into tiny pieces called packets, routing them across a physical infrastructure of cables and radio waves, and reassembling them at their destination—all in milliseconds.

  Q.2: What is request-response?
  ANS; Request:
   A message sent by a "client" (like your web browser or a mobile app) to a "server," asking for data, a file, or an action.
  Response:
   The server's reply back to the client, containing the requested information, an action confirmation, or an error message.

   Q.3: Frontend vs backend vs database?
   ANS; The frontend is what users see and click,
    the backend is the hidden brain that processes data, and
    the database is the storage vault where information is kept.

    Q.4: What is browser?
    ANS; A web browser is a software application that allows you to access,     navigate, and view information on the internet.

    Q.5: Why terminal exists?
    ANS; Terminals exist to provide a direct, text-based interface to a computer's operating system.

    Q.6: What happens when website opens?
    ANS; When you open a website, your browser translates the URL into a physical server address, establishes a secure connection, and downloads the page's files.

    Q.7: What DNS does?
    ANS; The Domain Name System (DNS) acts as the phonebook of the internet.

    Q.8: What request-response means?
    ANS; Request-response is a fundamental communication pattern where a client sends a message asking for information or an action, and a server processes it and sends back a reply.


    Q.9: Frontend vs backend?
    ANS; Front end is the visible, client-side part of a website or app that users interact with directly. Back end is the server-side, behind-the-scenes engine that processes data, manages business logic, and ensures everything functions properly.

    Q.10: What GitHub is?
    ANS; GitHub is a cloud-based platform where developers store, share, and collaborate on software code.


    Q.11: Why VS Code is used?
    ANS; Visual Studio Code (VS Code) is primarily used because it strikes a perfect balance between a lightweight, fast text editor and a feature-rich integrated development environment (IDE).

    Q.12: What terminal is?
    ANS; A terminal is a text-based interface used to interact with a computer's operating system.
